# Show HN: AIIR — Cryptographic receipts for commits with declared AI involvement

AI writes 30–50% of new code at most companies. Copilot, ChatGPT, Claude, Cursor, Windsurf, Devin — it all goes into `git commit` with no systematic record of what was human and what was machine.

Your auditors will eventually ask: *"Which code was AI-generated? Can you prove it?"*

We built **AIIR** (AI Integrity Receipts) to answer that question.

## What it does

AIIR scans git commits for AI authorship signals and generates content-addressed JSON receipts — cryptographic proof of what each commit contained and whether AI was involved.

```bash
pip install aiir
aiir --range HEAD~5..HEAD --pretty
```

One command. Zero dependencies. Apache 2.0.

## How it works

1. **Detection**: Scans commit metadata for 50+ AI signals across 10+ tools (Copilot, ChatGPT, Claude, Cursor, Windsurf, Devin, Gemini, Tabnine, Amazon Q, GitLab Duo)
2. **Receipt generation**: Produces a JSON receipt with commit metadata, AI attestation, and provenance
3. **Content addressing**: SHA-256 hash over canonical JSON of core fields — any modification invalidates the receipt
4. **Signing** (optional): Sigstore keyless signing for non-repudiation

A receipt looks like this:

```json
{
  "type": "aiir.commit_receipt",
  "schema": "aiir/commit_receipt.v1",
  "commit": {
    "sha": "c4dec85630...",
    "subject": "feat: add auth middleware",
    "author": {"name": "Jane Dev"},
    "files_changed": 4
  },
  "ai_attestation": {
    "ai_authored": true,
    "signals": [{"type": "trailer", "value": "Generated by: GitHub Copilot"}],
    "signal_count": 1
  },
  "content_hash": "sha256:a3f8b2c1...",
  "receipt_id": "g1-a3f8b2c1d4e5f6a7..."
}
```

## Why now

- **EU AI Act** transparency obligations are phasing in — general-purpose AI rules from August 2025, high-risk from August 2026
- **SOC 2 / ISO 27001** auditors are starting to ask about AI code provenance
- **Insurance underwriters** want evidence of AI involvement tracking
- There's currently no standard, open-source tool for this

## What's included

- **CLI**: `pip install aiir` — works on Python 3.9+
- **GitHub Action**: `uses: invariant-systems-ai/aiir@v1` — one line in your workflow
- **GitLab CI**: Native integration with MR comments, SAST reports, Duo detection
- **MCP Server**: 7 tools for Claude, Copilot, Cursor, Continue, Cline, Windsurf
- **Pre-commit hook**: Catch AI commits before they land
- **Policy engine**: Enforce AI authorship limits in CI (`strict` / `balanced` / `permissive`)
- **Sigstore signing**: Keyless, OIDC-based receipt signing
- **Release verification**: VSA (Verification Summary Attestation) — SLSA-style release gates
- **JS SDK**: Zero-dependency browser + Node.js verifier
